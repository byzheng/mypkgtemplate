# mypackagetemplate

This is a template for creating R packages with best practices and common configurations. It includes a sample `DESCRIPTION` file, a `CNAME` file for GitHub Pages, and a structure for organizing your package files.




## Usage

To use this template, simply copy the files and directories into your new package directory. Then, customize the `DESCRIPTION` file with your package's information, such as the package name, title, description, and URLs.




## Installation

Currently on [Github](https://github.com/byzheng/mypkgtemplate) only. Install with:

```r
remotes::install_github('byzheng/mypkgtemplate')
```
